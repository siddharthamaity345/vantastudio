# VANTA STUDIO

A premium creative studio landing page built with React 18, Vite, GSAP, and Lenis.

**Live URL:** https://vantastudio-five.vercel.app/

## Features

- **Kinetic typography** with GSAP ScrollTrigger animations
- **Custom cursor** with lerp smoothing and hover scaling
- **Image trail effect** on hero section (mouse-following project images)
- **Smooth scroll** via Lenis with GSAP sync
- **Web Audio API** synthesized sounds (ambient drone, hover tones, click pop)
- **Infinite marquee** with alternating directions
- **Floating decorative stickers** across sections
- **Responsive design** with mobile hamburger menu
- **Accessibility** support for `prefers-reduced-motion`

## Tech Stack

- **Framework**: React 18 (Vite)
- **Animation**: GSAP 3 + ScrollTrigger
- **Smooth Scroll**: Lenis
- **Fonts**: Bebas Neue + DM Sans (Google Fonts)
- **Audio**: Web Audio API (synthesized, no external files)
- **Icons**: Inline SVG only
- **Images**: picsum.photos placeholder URLs
